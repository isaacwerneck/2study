# 2study
Personal study tracker with login and backend storage.

## Setup
1. Open a terminal in this folder.
2. Run `npm install`.
3. Run `npm start`.
4. Open the URL shown in the terminal, typically `http://localhost:3000`.

## Notes
- The app now uses a local Node.js backend.
- If `3000` is already in use, the server will try the next available port.
- Accounts and tasks are saved in `database.json`.
- Login with username and password.
- All task data is stored per account.
